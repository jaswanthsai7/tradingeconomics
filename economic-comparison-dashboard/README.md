# 🌍 Global Economy Dashboard

A powerful yet simple dashboard to explore and compare country-level economic data using interactive visualizations and real-time API data.

## ✨ Features

- Country selection & comparison
- Key economic indicators (GDP, Inflation, etc.)
- Region filtering
- Interactive charts
- Responsive and modern UI
- Real-time data powered by TradingEconomics API

## 🧪 Tech Stack

- **Frontend:** Next.js (JavaScript)
- **Styling:** TailwindCSS
- **Data Source:** TradingEconomics API
- **Charts:** Chart.js / Recharts
